[[brandtDesigningExploratoryDesign2006]] describes these as Games to conceptualize Designing. With reference to [[1980? Habraken - Concept Design Games]] Aim is to study how designers manipulate and transform complex configurations, while making agreements and rules to how to go about their work. Purpose of these Games is to be a tool for research. 

The 9 Games from [[1980? Habraken - Concept Design Games]] create configurations using game pieces on a game board. Neither of these resemble real-life artifacts.

- [[Delta Game]]

Neither of these games include users.
